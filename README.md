## Fox - Cloud Server

Cloud Server.

## Run

```bash
$ git clone -b cloud-server --single-branch git@github.com:soulteary/fox.git cloud-server
$ cd cloud-server
$ npm start
```

## Lint Code

```bash
$ git clone -b cloud-server --single-branch git@github.com:soulteary/fox.git cloud-server
$ cd cloud-server
$ npm install
$ npm run eslint
```
